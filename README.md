# CidadaniaAtiva

## Portal de Notificações Cidadãs

Este projeto tem como objetivo desenvolver uma solução digital para facilitar a comunicação entre cidadãos e órgãos públicos, permitindo o registro e acompanhamento de problemas urbanos, como buracos, lixo acumulado e falhas na iluminação.

A aplicação contará com um back-end em Java com Spring Boot, um banco de dados MySQL e uma versão mobile desenvolvida em Flutter. O sistema permite o envio de denúncias com descrição e imagem, além da visualização e acompanhamento das ocorrências.

A proposta está alinhada aos Objetivos de Desenvolvimento Sustentável (ODS), contribuindo para cidades mais organizadas, participativas e eficientes por meio da tecnologia.

Desenvolvido por Victor Santana, Gustavo Marques e Nicoly Senna.
